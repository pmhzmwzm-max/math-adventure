设计原则

🎨 温暖俏皮

使用奶油色调和温暖色彩，避免冷冰冰的科技感

珊瑚橙作为主色传达温暖和鼓励，薄荷绿表示成功和进步，阳光黄用于奖励和庆祝。

⭕ 有机形状

超大圆角和不规则形状，营造友好亲和感

所有元素使用圆润边角，岛屿采用不规则有机曲线，消除任何锐利感。

✨ 微妙动效

流畅的弹性动画，带来愉悦的交互反馈

按钮有明显的按下反馈，输入框有呼吸光效，卡片有悬浮动画，整体感觉活灵活现。

📖 易读性优先

圆润字体配合高对比度，确保儿童轻松阅读

选择 Baloo 2 和 Nunito 字体，既可爱又专业。文字与背景保持足够对比度。

如何使用

应用到项目

  1. 将 :root 中的 CSS 变量复制到项目的 index.css
  2. 在 HTML <head> 中添加 Google Fonts 链接
  3. 根据组件样式更新 React 组件的 className
  4. 调整颜色变量以匹配设计系统
  5. 测试所有动画和交互效果
重要: 此设计系统已针对 3-8 岁儿童优化，通过了色彩对比度和可读性测试。
