## 六、题目生成规则

### 题目格式标准
```typescript
interface Question {
  id: number;
  type: 'choice' | 'input' | 'comparison' | 'sorting';
  question: string;        // 题目文本
  options?: string[];      // 选择题选项（4个）
  answer: string | number; // 正确答案
  hint?: string;           // 提示文字
  difficulty: 1 | 2 | 3;   // 难度等级
  visual?: string;         // 视觉元素（emoji或图片）
}
```

### 难度分布建议
- 简单（难度1）：40% —— 基础概念、直接计算
- 中等（难度2）：50% —— 需要思考、综合运用
- 困难（难度3）：10% —— 挑战题、应用题

### 视觉元素使用
- 幼儿园：大量使用 emoji 图案（🍎🌟🌈🐱等）
- 一年级：适度使用 emoji，结合数字
- 二年级：少量 emoji，以数学符号为主
- 三年级：以纯数字为主，少量图形辅助
