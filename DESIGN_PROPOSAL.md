## 四、QuizScreen (答题页面)

### 设计描述

**背景**：
- 主题色渐变背景
- 玻璃模糊层
- 浮动装饰光斑

**题目卡片**：
- 玻璃拟态效果 (Glassmorphism)
- 半透明白色背景 + 模糊
- 圆角矩形，大阴影

**输入框**：
- 活跃状态：脉冲发光动画（黄色光晕）
- 已填写：纯白背景
- 错误状态：红色背景 + 抖动动画

**连击显示**：
- 3连击：橙色火焰图标 + 黄→红渐变文字
- 5连击：粉色闪电图标 + 粉→橙渐变文字 + 更大粒子
- 10连击：紫色皇冠图标 + 紫→粉→红渐变文字 + 最大粒子爆发
- 粒子效果：多层彩色圆形向外扩散

**数字键盘**：
- 白色圆角按钮
- 悬停：轻微上浮 + 放大
- 按下：阴影收缩 + 缩小
- 删除键：红色图标

**宠物互动**：
- 每3连击宠物从右侧滑入
- 对话气泡（不同鼓励语）
- 上下浮动动画

## 五、动画规范

### 过渡曲线

```
快速: 150ms cubic-bezier(0.4, 0, 0.2, 1)
标准: 300ms cubic-bezier(0.4, 0, 0.2, 1)
缓慢: 500ms cubic-bezier(0.4, 0, 0.2, 1)
弹性: 500ms cubic-bezier(0.34, 1.56, 0.64, 1)
```

### 关键动画

```
1. 浮动：y: [0, -15, 0], duration: 3-5s, repeat: infinity
2. 脉冲：scale: [1, 1.1, 1], boxShadow: ["...", "...", "..."]
3. 粒子爆发：x/y 散射, scale: 0 → 1.5, opacity: 1 → 0
4. 错误抖动：x: [-5, 5, -5, 5, 0], duration: 0.4s
5. 弹簧进入：type: "spring", bounce: 0.5-0.7
```
